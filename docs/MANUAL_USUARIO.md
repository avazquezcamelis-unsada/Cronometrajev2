MANUAL DE USUARIO
Sistema de Gestión y Clasificación de Carreras Deportivas

============================================================
1. CREAR UNA CARRERA
============================================================

- Abrir módulo Carreras.
- Seleccionar Nueva Carrera.
- Ingresar:
    - Nombre
    - Fecha
    - Lugar
    - Distancias
- Guardar.

============================================================
2. INSCRIBIR UN CORREDOR
============================================================

- Seleccionar una carrera.
- Abrir Inscripciones.
- Completar:
    - DNI
    - Apellido
    - Nombre
    - Sexo
    - Ciudad
    - Fecha de nacimiento
    - Team
    - Distancia
    - Número
    - Talle

- Confirmar inscripción.

============================================================
3. INICIAR CARRERA
============================================================

- Seleccionar carrera.
- Presionar Iniciar Carrera.

El sistema registra automáticamente la hora de inicio.

============================================================
4. REGISTRAR LLEGADA
============================================================

El sistema permite registrar llegadas mediante:

- Registro automático.
- Registro manual.
- Registro mediante QR (si está habilitado).

Cada llegada queda almacenada junto con:

- Tiempo registrado.
- Fecha y hora de registro.
- Tipo de registro utilizado.
- Observaciones.

============================================================
5. CORREGIR TIEMPO MANUALMENTE
============================================================

- Seleccionar corredor.
- Ingresar tiempo corregido.
- Guardar.

El estado pasa a CORREGIDO.

============================================================
6. FINALIZAR CARRERA
============================================================

- Seleccionar carrera.
- Presionar Finalizar Carrera.

El sistema marca automáticamente los corredores
sin llegada registrada.

============================================================
7. CONSULTAR RESULTADOS
============================================================

- Abrir Resultados.
- Filtrar por:
    - Distancia
    - Sexo
    - Categoría

============================================================
8. CONSULTAR PODIOS
============================================================

- Abrir Podios.

El sistema genera automáticamente:

- Podios generales.
- Podios por categoría.
